# **App Name**: SavdoBot

## Core Features:

- AI Assistant Chat Interface: Act as an interactive AI assistant tool, understanding natural language commands (Russian, Uzbek) to manage transactions, debts, and provide financial guidance and insights through a conversational interface.
- Basic Financial Tracking: Record daily income and expenses for automatic profit calculation and a clear financial overview without complex accounting knowledge.
- Sales (POS) Management: Efficiently add new sales, select items from a customizable product/service catalog, and process transactions quickly.
- Debt Management: Track client debts, record payments, and maintain a history of all outstanding balances.
- Client Database: Maintain a comprehensive client database including purchase history and current debt status.
- Financial Reports & Analytics: Provide simple visual graphs and key performance indicators for daily, weekly, and monthly profits, and top-selling items, generated by an AI tool.
- Offline Mode with Sync: Enable seamless operation without an internet connection, with automatic data synchronization to the cloud once connectivity is restored.

## Style Guidelines:

- Primary brand color: A clean, professional blue (#268FDF) that signifies trust and efficiency, providing excellent contrast on a light background.
- Background color: A very light, desaturated bluish-grey (#ECF0F3) that supports a minimalist aesthetic and ensures high readability for text.
- Accent color: A vibrant, modern violet (#6354DB) chosen for calls-to-action and important highlights, creating dynamic visual interest while maintaining professionalism.
- Body and headline font: 'Inter' (sans-serif) for its neutral, objective, and highly legible characteristics, suitable for a clean, chat-focused interface across various screen sizes.
- Minimalist vector icons with a consistent outline style to maintain a clean, modern, and easily scannable interface.
- Responsive layout with a primary focus on the AI chat interface, ensuring intuitive navigation and clear display of financial data across web and mobile devices.
- Subtle and performant transition animations for smooth state changes and interactive elements, prioritizing speed and responsiveness on potentially weaker devices.
